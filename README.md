## üéØ Problem Statement
Many students and professionals struggle to find career paths that align with their unique profiles. Traditional counseling is often generic and expensive. **CareerNexus** solves this by leveraging data to provide:
1.  **Aptitude Assessment:** Identifying natural strengths.
2.  **Aspirations Mapping:** Aligning suggestions with long-term dreams.
3.  **Skill Gap Analysis:** Identifying what is missing.
4.  **Future Progression:** Predictive career trajectories.

---

## Key Features
*   **Neural Pathway Dashboard:** A comprehensive analysis of your career fit generated by AI.
*   **Neural Mentor Chat:** A context-aware chatbot that remembers your profile and previous conversations to give specific advice.
*   **Cognitive Assessment:** Analyzes problem-solving styles, team roles, and learning preferences.
*   **Privacy First:** All AI processing happens locally on your device using **Ollama**. No data is sent to the cloud.
*   **Modern UI:** A beautiful "Black/Gray Glassmorphism" interface with animated backgrounds.
*   **Profile Management:** Users can edit their education, skills, and goals dynamically.

## üóÑ Database Configuration

1.  Open your MySQL Client (e.g., phpMyAdmin or Workbench).
2.  Create a database named `career_nexus`.
3.  Run the following SQL commands to create the required tables:

```sql
CREATE DATABASE IF NOT EXISTS career_nexus;
USE career_nexus;

-- Users Table
CREATE TABLE IF NOT EXISTS users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    has_onboarded BOOLEAN DEFAULT FALSE
);

-- Profiles Table
CREATE TABLE IF NOT EXISTS profiles (
    user_id INT PRIMARY KEY,
    full_name VARCHAR(100),
    age INT,
    education VARCHAR(100),
    current_role VARCHAR(100),
    experience INT,
    interests TEXT,
    dream_goal TEXT,
    prob_solving VARCHAR(50),
    team_role VARCHAR(50),
    environment VARCHAR(50),
    learning_style VARCHAR(50),
    hard_skills TEXT,
    soft_skills TEXT,
    missing_skill TEXT,
    analysis_result LONGTEXT,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

## üß† AI Model Setup (Ollama)

This project uses a custom system prompt on top of Llama 3.

1.  **Start Ollama:** Open a terminal and run:
    ```bash
    ollama serve
    ```
2.  **Pull the Base Model:** Open a **new** terminal window and run:
    ```bash
    ollama pull llama3
    ```
3.  **Create the Custom Model:**
    *   Ensure the file named `Modelfile` is in your project directory (content provided in project source).
    *   Run the following command to create the **career-guru** model:
    ```bash
    ollama create career-guru -f Modelfile
    ```

##  Running the Application

1.  **Start the Backend Server:**
    ```bash
    python app.py
    ```
    *You should see: `Running on http://127.0.0.1:5000`*

2.  **Launch the Frontend:**
    *   Simply double-click `index.html` to open it in your browser.
    *   *Note: For best performance, use Chrome, Edge, or Firefox.*

3.  **Usage:**
    *   Register a new account.
    *   Complete the 3-Step Onboarding (Identity, Aptitude, Skills).
    *   Wait for the AI to generate your Dashboard.
    *   Chat with the Neural Mentor for specific advice!
